# GitHub actions for R w/ `{renv}` activated

## R-CMD-check-renv

```
usethis::use_github_action(
  url = "https://raw.githubusercontent.com/CorradoLanera/actions/master/R-CMD-check-renv.yaml"
)
usethis::use_github_actions_badge("R-CMD-check")
```
